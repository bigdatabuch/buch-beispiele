# Kapitel 12 - Kafka

Dieses Kapitel zeigt ein einfaches Beispiel für die Nutzung von Apache Kafka mit Python. 

Der Kafka-Broker wird über Docker Compose gestartet. Anschließend sendet ein Python-Producer Nachrichten an ein Kafka-Topic, während ein Python-Consumer diese Nachrichten empfängt und verarbeitet. Das Beispiel dient als Einstieg in eventbasierte Kommunikation. Es zeigt, wie Anwendungen Nachrichten über Kafka asynchron austauschen können.
