# Kapitel 10: Data Mesh

Data Mesh ist ein soziotechnischer Ansatz zur Architektur und Organisation analytischer Datenplattformen. Das Kapitel stellt die vier zentralen Prinzipien von Data Mesh vor.

## Beispiele

Das Kapitel enthält keine Code-Beispiele, sondern konzentriert sich auf die architektonischen und organisatorischen Aspekte von Data Mesh.

### Tool-Beispiel: Data Mesh Manager

Als praktisches Beispiel wird der **Data Mesh Manager** vorgestellt:
- Community Edition: https://github.com/datamesh-manager/datamesh-manager-ce
- SaaS-Variante: https://datamesh-manager.com

Der Data Mesh Manager unterstützt bei der Umsetzung der Data-Mesh-Prinzipien mit folgenden Funktionsbereichen:
- Data Product Inventory (Katalog für Datenprodukte)
- Data Contract Management (Lebenszyklus von Datenverträgen)
- Request- und Approval-Workflow (Self-Service für Datenzugriff)
