# Kapitel 6: Big Data-Management

Dieses Kapitel behandelt die Verwaltung großer Datenmengen und gibt einen Überblick über verschiedene Ansätze zur Datenspeicherung und -verwaltung.

## Beispiele

Dieses Kapitel enthält keine Code-Beispiele. Die konkreten Implementierungen der vorgestellten Konzepte finden sich in den darauf folgenden Kapiteln, beispielsweise:

- Kapitel 11: Key-Value Stores (Redis)
- Kapitel 11: Document Stores (MongoDB)
- Kapitel 11: Wide-Column Stores (Cassandra)
- Kapitel 11: Graph-Datenbanken (Neo4j, GraphDB)
- Kapitel 11: Time-Series-Datenbanken (InfluxDB)
- Kapitel 11: Vektor-Datenbanken (Milvus)
- Kapitel 11: Suchmaschinen (OpenSearch)
- Kapitel 11: Multi-Model-Datenbanken (ArangoDB)
- Kapitel 12: Data Warehouses und Data Lakes (mit Spark, DuckDB, Delta Lake)
